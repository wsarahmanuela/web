O CSS documento de estilzação em cascata criado em 1993, com sua primeira versão amplamente utiçizada em 199. A linguagem CSS permite definir regras que f=efinem caracteristicas para os elemntos HTML como cor, borda e comportamento, além de animações e transições.

Ao logo dos anos o css evolui e hoje temos o css 1 2 e 3, cada das versões adicionou novos recursos que eram demandadas no periodo.

  1. CSS 1 1996 - própriedade básicas de estilização e formatação, além de controle do posicionamento dos elemntos html com propriedades float e position
  2. CSS 2 2998 - adição de propriedades de animação e transição e melhoria do position
  3. CSS 3 2008 - incremnto gradual de novas funcionalidades do CSS com animação e ferramentas de posicionamento como display:flex (2008) e grid (2013

Uma regra css pode ser definida com a seguinte sintaxe

    Seletor{
        propriedade1: Valor;
        propriedade2: Valor;
    }

Podemos incluir códigos CSS de 3 formas diferentes

CSS externo
Quando definismos as regras CSS a serem utilizados por meio de um arquivo externo.
CSS incorporado
Quando definimos as regras CSS por meio da tag <style >
CSS inline
Quando definimos código css de estilização no proprio elemnto html como: Texto de cor vermelha com css inline
