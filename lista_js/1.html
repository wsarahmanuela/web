O que é JavaScript?

JavaScript é uma linguagem de programação criada inicialmente para permitir que páginas web interagissem com o usuário, proporcionando uma experiência dinâmica. A motivação para sua criação surgiu nos anos 90, quando os navegadores precisavam de mais funcionalidades do que apenas mostrar informações estáticas. Existem duas formas principais de execução do JavaScript: no navegador e no servidor. No navegador, o JavaScript é executado no lado do cliente, permitindo interação com o HTML e o CSS da página sem precisar de interação com um servidor. No servidor, o JavaScript é utilizado para criar aplicações web, como com o Node.js, permitindo manipulação de banco de dados, servidores e outras funcionalidades de backend. O conceito de "sandbox" no navegador se refere a um ambiente restrito onde o código JavaScript é executado. Isso significa que o código não pode acessar diretamente o sistema de arquivos do usuário ou outras partes do computador, oferecendo segurança contra possíveis ataques maliciosos. Alguns objetos globais comumente utilizados no JavaScript incluem: document: permite manipular o conteúdo da página web. console: utilizado para registrar mensagens no console, útil para depuração. alert(): exibe uma mensagem simples em uma caixa de diálogo na tela do usuário.
