Lista com comando basicos o git

  • git init - Cria um repertorio em uma pasta vazia
  • git clone urlrespositorio - Clona um repositorio
  • git diff - exibe a diferença entre versões do repositorio
  • git add - Adiciona modificaçoes do wk no stage
  • git status - Mostra o estado atual do repertorio, modificaçoes arquirvos versionados
  • git commit - Cria um commit no repertorio LOCAL
  • git push - Envia as modificaçoes do repertorio LOCAL para o remoto
  • git log - Exibe no terminal od commits do repositorio na branch atual
  • git log --oneline -- all - Exibe todos os commits de forma resumida
  • git branch - Lista as branches do repositorio
  • git chechout - Muda de branch
  • git checkout -b - Cria e muda para uma nova branch
